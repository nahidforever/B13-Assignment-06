# 🎯 DigiTools – Digital Tools Buying Website

## 📌 Project Overview

**DigiTools** is a modern digital tools marketplace where users can browse premium digital products and add them to a cart. Users can explore tools, view features, and simulate a simple checkout experience.

The website is designed based on Figma and built using React with a clean and responsive UI.

## 🚀 Technologies Used

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌼 DaisyUI
- 🧠 JavaScript (ES6+)
- 🔔 React-Toastify
- 📦 JSON (for product data)

## ✨ Key Features

### 🛒 1. Dynamic Cart System

- Add products to cart instantly
- Remove individual products
- Proceed to checkout clears the cart
- Real-time cart count update in Navbar

### 🔄 2. Product & Cart Toggle System

- Toggle between Products and Cart view
- Default view shows product list
- Empty cart message displayed when no items are added

### 📦 3. Product Display with Rich UI

- 3-column responsive product layout
- Each card includes:
  - Name, Price, Description
  - Features list
  - Tag (Popular, New, Best Seller)
  - Icon
  - Buy Now button
- Clean UI with icons and gradient design

### 🎨 Additional Features

- 🔔 Toast notifications (Add, Checkout)
- 📱 Fully responsive design
- 🪜 Steps section for user guidance
- 💰 Pricing section with modern UI
- 🦶 Footer and Navbar designed based on Figma
